\section{Word-Like Formatting Features}

\subsection{Text Styling}
LaTeX makes basic text formatting simple:
\begin{itemize}
    \item \textbf{This text is Bold} (Command: \texttt{\textbackslash textbf\{...\}})
    \item \textit{This text is Italic} (Command: \texttt{\textbackslash textit\{...\}})
    \item \textbf{\textit{This text is Bold and Italic}} 
    \item \underline{This text is Underlined} (Command: \texttt{\textbackslash underline\{...\}})
\end{itemize}

\subsection{Text Sizes}
You can change the font size relative to the document default (11pt). Note that these commands affect all text following them until you close the group (curly braces).

\begin{itemize}
    \item {\tiny This is tiny text} \hfill (\texttt{\textbackslash tiny})
    \item {\scriptsize This is scriptsize text} \hfill (\texttt{\textbackslash scriptsize})
    \item {\footnotesize This is footnotesize text} \hfill (\texttt{\textbackslash footnotesize})
    \item {\small This is small text} \hfill (\texttt{\textbackslash small})
    \item {\normalsize This is normalsize text (Default)} \hfill (\texttt{\textbackslash normalsize})
    \item {\large This is large text} \hfill (\texttt{\textbackslash large})
    \item {\Large This is Large text} \hfill (\texttt{\textbackslash Large})
    \item {\LARGE This is LARGE text} \hfill (\texttt{\textbackslash LARGE})
    \item {\huge This is huge text} \hfill (\texttt{\textbackslash huge})
    \item {\Huge This is Huge text} \hfill (\texttt{\textbackslash Huge})
\end{itemize}
